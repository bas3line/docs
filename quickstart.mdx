## Install

```bash
cargo install ultrabalancer
```

Or download from [releases](https://github.com/bas3line/ultrabalancer/releases)

## Run Your First Load Balancer

```bash
# Basic - 2 backends, round-robin
ultrabalancer -b server1:8080 -b server2:8080

# Custom port
ultrabalancer -p 80 -b 192.168.1.10:8080 -b 192.168.1.11:8080

# With weights
ultrabalancer -b app1:8080:200 -b app2:8080:100 -b app3:8080:50
```

## Add Monitoring

```bash
# One command for Grafana + Prometheus
ultrabalancer dashboard --start
```

- Grafana: http://localhost:3000 (admin + auto-generated password)
- Prometheus: http://localhost:9090

## Using a Config File

Create `config.yaml`:

```yaml
listen_address: "0.0.0.0"
listen_port: 8080
algorithm: "round-robin"

backends:
  - host: "192.168.1.10"
    port: 8080
    weight: 100
  - host: "192.168.1.11"
    port: 8080
    weight: 100

health_check:
  enabled: true
  interval_ms: 5000
  max_failures: 3
```

Run it:

```bash
ultrabalancer -c config.yaml
```

## Test It

```bash
# Send some requests
curl http://localhost:8080
curl http://localhost:8080/health
```
