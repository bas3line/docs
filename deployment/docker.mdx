## Quick Start

```bash
# Run with backends
docker run -d -p 80:8080 \
  ghcr.io/bas3line/ultrabalancer:latest \
  start round-robin backend1:8080 backend2:8080
```

## With Config File

```bash
# Create config
cat > config.yaml << 'EOF'
listen_address: "0.0.0.0"
listen_port: 8080
algorithm: "round-robin"
backends:
  - host: "backend1"
    port: 8080
  - host: "backend2"
    port: 8080
EOF

# Run with config
docker run -d -p 80:8080 -v $(pwd)/config.yaml:/config.yaml:ro \
  ghcr.io/bas3line/ultrabalancer:latest \
  start -c /config.yaml
```

## Docker Compose

```yaml
services:
  ultrabalancer:
    image: ghcr.io/bas3line/ultrabalancer:latest
    ports:
      - "80:8080"
    volumes:
      - ./config.yaml:/config.yaml:ro
    command: start -c /config.yaml

  backend1:
    image: nginx:alpine

  backend2:
    image: nginx:alpine
```

```bash
docker compose up -d
```

## Environment Variables

```bash
docker run -d -p 80:8080 \
  -e ULTRA_ALGORITHM=least-connections \
  -e ULTRA_BACKENDS=backend1:8080,backend2:8080 \
  ghcr.io/bas3line/ultrabalancer:latest
```

## Health Check

```yaml
services:
  ultrabalancer:
    image: ghcr.io/bas3line/ultrabalancer:latest
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:8080/metrics"]
      interval: 30s
      timeout: 10s
      retries: 3
```
